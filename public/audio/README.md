# 音频占位

此目录用于存放游戏音频文件。

## 支持格式
- MP3 (推荐)
- OGG
- WAV

## 建议文件命名
- bgm_menu.mp3 - 主菜单背景音乐
- bgm_game.mp3 - 游戏进行背景音乐
- bgm_result.mp3 - 结算画面音乐
- sfx_collect.mp3 - 收集花瓣音效
- sfx_craft.mp3 - 合成成功音效
- sfx_button.mp3 - 按钮点击音效
- sfx_success.mp3 - 稀有物品获取音效
- sfx_awaken.mp3 - 恋人唤醒音效

当前版本使用 Web Audio API 程序化生成简单音效，无需额外音频文件即可运行。
