# 图片资源占位

此目录用于存放自定义图片资源。

## 当前游戏使用动态纹理生成

所有图片纹理均由 Phaser Graphics API 在运行时生成，无需额外图片文件。

## 如需自定义资源

可放置以下文件（文件名对应纹理 key）：

### 花瓣 (建议 128x128 PNG)
- petal_rose.png
- petal_lily.png
- petal_lavender.png
- petal_sunflower.png
- petal_jade.png
- petal_starlight.png
- petal_moonstone.png
- petal_dream.png
- petal_aurora.png
- petal_eternal.png

### 角色 (建议 128x128 ~ 256x256 PNG)
- player.png
- lover_sleeping.png
- lover_awake.png

### UI (建议 PNG)
- btn_normal.png (240x80)
- btn_hover.png (240x80)
- btn_disabled.png (240x80)
- btn_small.png (120x56)

### 背景 (建议 750x1334 PNG)
- bg_forest.png

### 粒子
- particle.png (32x32)
- sparkle.png (64x64)
